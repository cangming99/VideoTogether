# 面板 UI 现代化改造设计

## 概述

为 VideoTogether 面板设计现代化的 UI，由于增加了聊天记录功能，需要更大的面板空间。面板有两个模式：全屏模式和非全屏模式，本文聚焦非全屏模式的改造。

## 设计选择

- **风格**: 轻量现代（Tailwind 风格）
- **布局**: 上下布局（聊天区域在上，按钮在底部）
- **昵称入口**: Header 右侧下拉菜单

## 面板规格

| 属性 | 当前值 | 新值 |
|------|--------|------|
| 宽度 | 260px | 280px |
| 高度 | 210px | 420px |
| 圆角 | 10px | 16px |

## 布局结构（从上到下）

```
┌─────────────────────────────────────┐
│ 📺 VideoTogether  [👤 小明 ▼][_][⚙️][❤] │  <- Header (36px)
├─────────────────────────────────────┤
│  👤 角色    👥 成员数                │  <- 状态栏 (~40px)
│  📋 房间: xxxxxx                    │
├─────────────────────────────────────┤
│  [聊天消息区域 - 可滚动]             │  <- 聊天区 (~200px)
│  - 自己的消息靠右（浅绿 #e8f5e9）    │
│  - 他人消息靠左（浅灰 #f0f2f5）      │
├─────────────────────────────────────┤
│  [输入框....................][发送]  │  <- 输入区 (~50px)
├─────────────────────────────────────┤
│  [创建房间]    [加入房间]            │  <- 按钮组 (~80px)
│  [退出房间]    [语音通话]            │
└─────────────────────────────────────┘
```

## Header 设计

- 高度：36px
- 左侧：Logo + "VideoTogether" 文字
- 右侧：昵称下拉菜单、最大化按钮、设置链接、捐赠链接
- 昵称下拉菜单：点击展开，显示"编辑昵称"和"取消"选项

## 颜色系统

| 用途 | 颜色 |
|------|------|
| 主色（按钮） | #1890ff |
| 成功色 | #23d591 |
| 危险色 | #ff4d4f |
| 背景 | #ffffff |
| 文字 | #000000d9 |
| 边框 | #e9e9e9 → 移除，改用阴影 |

| 他人气泡 | #f0f2f5 |

## 阴影

```css
box-shadow: 0 4px 20px rgba(0,0,0,0.08);
```

## 组件样式

### 按钮
- 圆角：8px
- 高度：32px
- padding: 4px 15px
- hover: 背景色加深，transition 0.3s

### 输入框
- 圆角：8px
- border: 1px solid #e9e9e9
- focus: border-color #1890ff, box-shadow

### 聊天消息
- 圆角：12px
- padding: 8px 12px
- max-width: 85%
- 自己消息：右对齐，浅绿背景
- 他人消息：左对齐，浅灰背景

### 昵称下拉菜单
- 点击昵称展开菜单
- 菜单位于 Header 下方
- 选项：编辑昵称（显示输入框）、取消

## 功能完整性

所有现有功能必须保留：
- 房间创建/加入/退出
- 语音通话
- 聊天消息发送/接收
- 聊天历史加载
- 气泡通知（最小化时）
- 设置/捐赠链接
